# UniVea – Student Services Platform

UniVea is a web-based platform designed to help students easily discover and access various academic and campus-related services. The platform connects students with service providers such as tutors, project guides, printing services, accommodation providers, and more in a simple and organized way.

The goal of UniVea is to create a centralized hub where students can quickly find trusted services that support their academic work and everyday student life.

---

## Features

- Clean and student-friendly user interface
- Search functionality to quickly find services
- Dedicated **Providers Page** listing available service providers
- Simple navigation for better user experience
- Modern and minimal UI design
- Responsive layout for different screen sizes
- Organized categories of student services

---

## Services Provided

UniVea connects students with various useful services, including:

### Academic Services
- Private tutors
- Assignment help
- Project guidance
- Programming help
- Study materials

### Student Utility Services
- Printing and photocopy services
- Stationery providers
- Laptop repair services

### Living & Campus Services
- PG / hostel accommodation
- Room rentals
- Food and tiffin services
